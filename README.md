## -  O que é a ferramenta conda, e qual a sua utilidade no desenvolvimento de programas em Python

A ferramenta Conda possibilita instalar, atualizar, bem como remover pacotes facilmente. Desta forma, faz com que que todas as dependências sejam resolvidas. Além disso, esta ferramenta permite que o usuário consiga inserir pacotes novos em ambientes isolados distintos, sendo que um ambiente não irá afetar o outro, promovendo alternação por meio dos ambientes gerados de forma prática e rápida. Ainda, quando há compartilhamento de máquina com usuários distintos, se torna mais pŕatico requisitar  instalações ao usuário administrador do sistema, sua maior utilidade no Python é que permite que cada projeto possua seu próprio ambiente, evitando conflitos e dificuldades na compatibilidade, sendo Python dependente ocorre a redução dos problemas no desenvolvimento de software.

## - Como funciona a ferramenta ACPred? Qual a sua finalidade?

A ferramenta ACPred é utilizada para predição de peptídeos anticâncer, foi desenvolvida a partir da utilização de características distintas de peptídeos e algoritmos de máquina dando enfâse ao Support Vector Mchine e Random Forest. Além disso, promove insights dos mecanismos de ação de peptideos anticâncer a partir dos recursos interpretáveis primordiais na identificação e análise profunda no que se refere a propriedades bioquímicas e biofísicas das atividades anticâncer dos peptídeos.

O front-end baseia-se na apresentação de um site, isto é, cuida das partes visuais, além da interação entre usuários e tela, gerencia e usabilidade, promovendo o design e a experiência do usuário. ## - O que são testes unitários? Qual a sua importância no desenvolvimento de software?

O intuito principal dos testes unitários é averiguar o funcionamento correto de unidades individuais do código, sendo assim "isolada" a unidade a ser testada, promovendo entradas exclusivas a fim de verificar se é a saída necessária. Definindo como "menor parte testável de um programa". Sua importância primordial no desenvolvimento de software é que a partir destes testes torna-se possível identificar e corrigir problemas, antes de realizar o lançamento do software, promovendo maior confiabilidade ao software, além de alertarem erros prováveis ou celunas para consigam gerar problemas futuros.
